# Sistema gerador de orçamentos em PDF e gerenciador de clientes e orçamentos.

Um programa desktop feito para otimizar a criação de orçamentos e gerenciar um banco de dados de clientes locais (pessoa física e jurídica).

## Demonstração

(https://youtu.be/bb5mB4T-i7c)

## Funcionalidades

- **Gestão de Clientes (CRUD básico):** Cadastro e edição de clientes físicos (CPF) e jurídicos (CNPJ).
- **Persistência Local:** Dados salvos de forma segura e offline utilizando **SQLite**.
- **Geração Automática de Documentos:** Exportação de orçamentos formatados e prontos para impressão em formato **PDF**.

## Tecnologias Utilizadas

- **Linguagem:** C#
- **Framework:** .NET 8 (Windows Forms)
- **Banco de Dados:** SQLite (com biblioteca `Microsoft.Data.Sqlite`)
- **Geração de PDF:** QuestPDF
- **Deploy:** Visual Studio Installer Projects
